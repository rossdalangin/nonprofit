# CausePro WordPress Theme

CausePro is a heartfelt, trustworthy, and action-oriented WordPress theme designed specifically for non-profit and charitable organizations. It is optimized to help you tell your story, engage supporters, and maximize online donations.

## Features

*   **Customizer-driven Homepage**: Easily build and manage your homepage sections directly from the WordPress Customizer.
*   **Custom Post Types**: Dedicated post types for "Causes" and "Events" to keep your content organized.
*   **Donation-focused Design**: A persistent and prominent "Donate Now" button in the header to drive conversions.
*   **Featured Campaign Section**: Highlight your most important fundraising campaign with a progress bar.
*   **Responsive & Accessible**: Designed to work beautifully on all devices and built with accessibility best practices in mind.

## Installation

1.  Download the theme as a `.zip` file.
2.  In your WordPress admin dashboard, go to **Appearance > Themes**.
3.  Click the **Add New** button at the top.
4.  Click the **Upload Theme** button.
5.  Choose the `CausePro.zip` file you downloaded and click **Install Now**.
6.  Once installed, click **Activate**.

## Theme Setup

After activating the theme, follow these steps to get your site set up.

### 1. Set Up the Homepage

The homepage is controlled by a page template and the WordPress Customizer.

1.  Go to **Pages > Add New**.
2.  Give the page a title, such as "Home".
3.  In the **Page Attributes** box on the right, select the **Homepage** template from the dropdown menu.
4.  Click **Publish**.
5.  Now, go to **Settings > Reading** in your WordPress admin.
6.  Set "Your homepage displays" to **A static page**.
7.  For the "Homepage" dropdown, select the "Home" page you just created.
8.  Click **Save Changes**.

### 2. Configure the Homepage Content

All content and sections on the homepage are managed via the Customizer.

1.  Go to **Appearance > Customize**.
2.  You will see a panel called **Homepage Sections**. This is where you control all the content for the homepage.
    *   **Section Order & Visibility**: Drag and drop the section names (hero, impact, causes, campaign, events) to reorder them on the page. Remove a name to hide that section.
    *   **Hero Section**: Set the main background image, the powerful headline, and the button text.
    *   **Our Impact Section**: Configure the four statistics that show your organization's impact. You can set a number, a label, and an icon (using [WordPress Dashicons](https://developer.wordpress.org/resource/dashicons/)).
    *   **Current Causes Section**: Set the headline and choose how many recent "Cause" posts to display.
    *   **Campaign Feature Section**: Customize the content for your featured campaign, including the fundraising goal and amount raised to power the progress bar.
    *   **Upcoming Events Section**: Set the headline and choose how many upcoming "Event" posts to display.

### 3. Configure Theme Options

Global settings for the theme are in **Appearance > Customize > Theme Options**.

*   **Donation Link**: Set the single, global URL for all primary "Donate Now" buttons. This is the most important setting!
*   **Colors**: Change the theme's primary accent color.
*   **Footer**: Update the copyright text shown in the site footer.

### 4. Add Causes and Events

*   **Causes**: Go to **Causes > Add New** in the left-hand menu to create a new cause. Add a title, a description, and a featured image.
*   **Events**: Go to **Events > Add New** to create a new event. In addition to the title and description, you can add:
    *   **Event Details**: Set the date/time, location, and a registration or info link.
    *   **Event Type**: Assign a type to the event, like "Fundraiser" or "Volunteer Opportunity".

### 5. Create the Causes and Events Archive Pages

1.  Go to **Pages > Add New** and create a page titled "Our Causes".
2.  Under **Page Attributes**, select the **Causes Page** template and publish.
3.  Go to **Pages > Add New** and create another page titled "Upcoming Events".
4.  Under **Page Attributes**, select the **Events Page** template and publish.
5.  You can now add these pages to your navigation menu under **Appearance > Menus**. The Events page will automatically display upcoming events and include a dropdown to filter them by type.
